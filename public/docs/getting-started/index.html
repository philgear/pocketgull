Getting Started

Developer Profile: g.dev/philgear Repository: github.com/philgear/pocketgull

Prerequisites

  • Node.js OpenJS Foundation JavaScript runtime built on Chrome's V8 engine. Originally created by Ryan Dahl in 2009. Pocket Gull requires v20+. nodejs.org → Click to explore with AI ≥ 20.0.0
  • npm
  • A Gemini API key Google DeepMind Personal API key from Google AI Studio. Required for live AI consult, multi-agent analysis, and chat features. Get API Key → Click to explore with AI (for live AI features)

Installation

# Clone the repository
git clone https://github.com/philgear/pocketgull.git
cd pocket-gull

# Install dependencies
npm install

Configuration

Pocket Gull’s backend natively parses .env files on boot. Create a .env file in the root directory to enable the DICOM fetcher and define your API keys:

GEMINI_API_KEY="your-gemini-key"
GOOGLE_CLOUD_PROJECT="your-gcp-project-id"
HC_DATASET="your-healthcare-dataset-id"
HC_DICOM_STORE="your-dicom-store-id"

Development

# Start the development server
npm run dev

The application will be available at http://localhost:4200. Enter your Gemini API key on the splash screen, or press Try Demo to load a pre-sampled patient with example AI analysis outputs.

Production Build

# Build for production
npm run build

# Preview the production build locally
npm run preview

The production build outputs to dist/ with full SSR Google · Angular Team Server-Side Rendering — Angular renders the initial page on the Express.js server, then hydrates client-side. Faster first paint and SEO. SSR Guide → Click to explore with AI support.

Deployment

Pocket Gull is architecturally designed to deploy to Google Cloud Run Google Cloud Fully managed serverless container platform. Auto-scales from zero. Pocket Gull deploys as a Docker container with Express.js + Angular SSR. Cloud Run Docs → Click to explore with AI .

Live Deployment

  • Primary URL: pocketgull.app
  • Cloud Run URL: understory-315235665910.us-west1.run.app

Automated Deployment Script

The repository includes a one-click deployment script at scripts/deploy.sh:

#!/usr/bin/env bash
set -euo pipefail

gcloud builds submit --tag gcr.io/$PROJECT_ID/pocket-gull
gcloud run deploy pocket-gull \
  --image gcr.io/$PROJECT_ID/pocket-gull \
  --platform managed \
  --region us-west1 \
  --allow-unauthenticated
