## Installation

```bash
# Clone the repository
git clone https://github.com/philgear/pocket-gull.git
cd pocket-gull

# Install dependencies
npm install
```

---

## Configuration

Pocket Gull's backend natively parses `.env` files on boot. Create a `.env` file in the root directory to enable the DICOM fetcher and define your API keys:

```bash
GEMINI_API_KEY="your-gemini-key"
GOOGLE_CLOUD_PROJECT="your-gcp-project-id"
HC_DATASET="your-healthcare-dataset-id"
HC_DICOM_STORE="your-dicom-store-id"
```

---

## Development

```bash
# Start the development server
npm run dev
```

The application will be available at `http://localhost:4200`. Enter your Gemini API key on the splash screen, or press **Try Demo** to load a pre-sampled patient with example AI analysis outputs.

### Automated Deployment Script

The repository includes a one-click deployment script at `scripts/deploy.sh`:

```bash
#!/usr/bin/env bash
set -euo pipefail

gcloud builds submit --tag gcr.io/$PROJECT_ID/pocket-gull
gcloud run deploy pocket-gull \
  --image gcr.io/$PROJECT_ID/pocket-gull \
  --platform managed \
  --region us-west1 \
  --allow-unauthenticated
```
