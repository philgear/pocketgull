# Changelog

All notable changes to Pocket Gull are documented here.
Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## v1.0.0-rc3 — 2026-06-08

**Porkbun DNS Automation, Enterprise EHR Blueprints & Security Hardening**

### Added
- **Porkbun DNS Automation Integration**: Created `scripts/update-porkbun-dns.js` to automatically sync Cloud Run custom domains (`pocketgull.app`) via the Porkbun API.
- **Enterprise Scaling & Valuation Manuals**: Added deep architectural blueprints in `docs/valuation_and_positioning.md` and `docs/study/src/pages/positioning.mdx` explaining BigQuery partitioning/clustering, Vertex AI context caching optimization, and HL7 FHIR integrations with OpenEMR/OpenMRS.
- **Dependency Vulnerability Resolution**: Patched package-lock overrides for `uuid` and `protobufjs` to achieve 0 vulnerabilities in the `pocketgull_api` sub-project.
- **Windows Pre-Commit Unicode Fix**: Resolved standard encoding console crashes in `scripts/phi_compliance_scanner.py` on Windows Anaconda environments.

## v0.9.5 — 2026-05-20

**Method of Loci Palace, Smiley Gesture Authentication, & AVS Therapy Suite**

### Added
- **Method of Loci (Clinical Memory Palace)**: Integrated an interactive 3D-assisted Method of Loci learning framework into the patient assessment report space, enabling clinicians to visually anchor complex care plan protocols and diagnostics to spatial memory hubs (Clinical Loci).
- **Smiley Gesture Authentication & WebAuthn Safeguard**: Replaced PIN-based logins with a canvas-based smiley drawing unlock flow in `SecureSplashComponent`. Integrated conditional platform checks to only present the biometric WebAuthn security button on supported devices.
- **AVS (Acoustic/Visual/Sensory) Therapy & Guided Circadian UI**: Added a clinic/patient visual entrainment interface featuring guided box breathing cycles and light frequency modulation, integrated natively within the system's tour guides and onboarding flows.
- **Comprehensive Third-Party Register**: Formulated `DEPENDENCIES.md` and the interactive `dependencies.mdx` documentation page, compiling licensing info for Angular, FastAPI sidecar, and Flutter apps to enforce absolute open-source compatibility.

## v0.9.0 — 2026-05-18

**Multilingual Care Plans & Orthomolecular Profiling**

### Added
- **Multilingual & Pediatric Care Plans**: Expanded the `translateReadingLevel` infrastructure across all providers to support Spanish, German, French, Mandarin, Dyslexia, and Pediatric formats.
- **Security & Red Teaming Framework**: Implemented `SECURITY.md` and a Vitest automated red-teaming suite (`tests/safety.spec.ts`) to actively verify safety boundaries.
- **Transparency Metadata**: Added `humans.txt`, `robots.txt`, `llms.txt`, and `sources.txt` to the public directory.
- **Cloud Build Integration**: Created `cloudbuild.yaml` to automate Docker builds and Google Cloud Run deployment pipelines.
- **Advanced Clinical Mock Data**: Expanded `MOCK_PATIENTS` with updated 2026 timestamps, Orthomolecular Biomarkers, and dedicated Pediatric and Multilingual test profiles.
- **Animal Comfort Protocols**: Engineered a Web Audio API synthesizer (`PetAuditoryService`) within the Care Plan Export module to generate species-specific soothing frequencies (Feline Purr at 25Hz, Canine Heartbeat at 60 BPM) for pets left home during hospitalizations.
